# Django-website.

Website for creating invoices and getting info about them on webpage with simplest css-style or by API through token access.

## How to run.

Firstly, install requirements.
```
pip install -r requirements.txt
```

The standard process for starting a Django built-in web server.
```
python manage.py runserver
```

Done. You can see result on <a href='http://127.0.0.1:8000/'>http://127.0.0.1:8000/</a> and browsable API page on <a href='http://127.0.0.1:8000/api/'>http://127.0.0.1:8000/api/</a>. For creating the invoice you need to log in by using admin account
